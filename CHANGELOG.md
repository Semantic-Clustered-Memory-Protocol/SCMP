# Changelog

All notable changes to SCMP will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.1.0] - 2026-02-13

### Added
- Initial open source release
- HNSW indexing for fast similarity search
- Multi-tier storage architecture (HOT/WARM/COLD)
- Vector quantization (int8, float16)
- IndexedDB persistence layer
- Automatic memory pressure monitoring
- Journal-based crash recovery
- Semantic clustering capabilities
- Ollama integration for embeddings
- Comprehensive TypeScript definitions
- Example applications (basic, RAG, chatbot)

### Features
- Cosine similarity search
- Batch operations support
- Metadata filtering
- Temporal decay
- Intelligent pruning
- Crash recovery
- Auto-save functionality
- Memory statistics and monitoring
